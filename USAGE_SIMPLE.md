# 올리브영 크롤러 간단 사용법

## 🚀 시작하기

### 1단계: 실행
```bash
cd /Users/changhyunpark/Documents/Oliveyoung
source venv/bin/activate
streamlit run app_v2.py
```

브라우저가 자동으로 열립니다 (http://localhost:8501)

## 🎯 사용 방법

### 방법 1: 키워드로 검색
1. 검색어 입력 (예: "한율 달빛유자")
2. "검색 및 크롤링 시작" 버튼 클릭
3. 완료될 때까지 대기 (30초~1분)
4. 결과 확인 및 폴더 열기

### 방법 2: URL 직접 입력
1. 올리브영 상품 페이지 URL 복사
2. URL 입력란에 붙여넣기
3. "URL로 크롤링 시작" 버튼 클릭
4. 완료될 때까지 대기
5. 결과 확인 및 폴더 열기

## ⚙️ 설정 옵션

### 브라우저 표시
- **체크 안 함 (기본)**: 브라우저가 보이지 않음, 빠름 ⚡
- **체크함**: 브라우저가 화면에 보임, 디버깅에 유용 🔍

**언제 켜야 하나?**
- 이미지 추출이 실패할 때 → 체크
- 문제 없이 잘 작동할 때 → 체크 해제

### 저장 형식
- **both (추천)**: JSON + CSV 둘 다 저장
- **json**: JSON만 저장 (프로그래밍용)
- **csv**: CSV만 저장 (엑셀용)

## 📁 결과 확인

크롤링 완료 후 `data/상품명_날짜시간/` 폴더에 저장됩니다:
```
data/
└── 한율_달빛유자_비타민_톤업팩폼_20251112_153045/
    ├── product_detail_merged.jpg  ← 이것! (병합된 이미지)
    ├── product_info.json
```

"📁 폴더 열기" 버튼을 클릭하면 자동으로 폴더가 열립니다.

## ❓ 자주 묻는 질문

### Q: "시작" 버튼을 먼저 눌러야 하나요?
**A: 아니요!** 이제 자동으로 브라우저가 시작됩니다. 바로 검색하세요.

### Q: 브라우저 표시는 언제 켜야 하나요?
**A: 디버깅할 때만 켜세요.**
- 평소: 체크 해제 (빠르고 조용함)
- 문제 발생: 체크 (뭐가 문제인지 확인)

### Q: 이미지 추출이 실패해요
**A: 다음을 시도해보세요:**
1. 브라우저 표시 체크
2. 다시 크롤링 시작
3. 브라우저 창에서 뭐가 문제인지 확인
4. 콘솔에 출력되는 에러 메시지 확인

### Q: 여러 상품을 한 번에 크롤링할 수 있나요?
**A: 현재는 한 번에 하나씩만 가능합니다.**
여러 개 하려면 반복해서 클릭하세요.

### Q: 크롤링한 이미지는 어디에 저장되나요?
**A: `data/` 폴더 안에 상품명_날짜시간 폴더로 저장됩니다.**

## ⚠️ 문제 해결

### "이미지를 찾을 수 없습니다"
1. 브라우저 표시 체크
2. 다시 시도
3. 콘솔에 `debug_page_source.html` 생성됨
4. 해당 파일 열어서 페이지 구조 확인

### "더보기 버튼을 찾을 수 없습니다"
- 정상입니다! 이미 펼쳐져 있을 수 있음
- 이미지 추출은 계속 진행됩니다

### 브라우저가 안 보여요 (브라우저 표시 체크했는데)
- 크롤러 "종료" 버튼 클릭
- 다시 크롤링 시작
- 새로 초기화될 때 설정이 적용됩니다
